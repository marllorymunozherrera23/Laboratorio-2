HOSTING/DOMINIO/WIX/GOOGLE SITES

HOSTING

Un hosting es un servicio de alojamiento web que te permite publicar un sitio web o aplicación en Internet. Cuando contratas un servicio de hosting, básicamente alquilas un espacio en un servidor físico donde puedes almacenar todos los archivos y datos necesarios para que tu sitio web funcione correctamente.

DOMINIO

Un nombre de dominio es la dirección que las personas escriben en un navegador para llegar a un sitio específico. Al igual que una huella digital, cada nombre de dominio es exclusivo de un sitio web y no se puede compartir ni duplicar con otras páginas web. Si observas la parte superior de esta página, verás que el nombre de dominio de este sitio web es "wix.com".

WIX

WIx es una plataforma de desarrollo web que basa su sistema en la nube, en la cual puedes hacer tu sitio web sin tener conocimientos profundos en programación. Te ofrece cientos de plantillas y funciones con las cuáles podrás jugar para crear tu sitio web, blog o tienda online. Lo único que debes hacer es registrarte en la plataforma y responder un par de preguntas que el sitio realiza antes de que puedas elegir el tipo de plantillas que deseas implementar.

GOOGLE SITES

Con Google Site puedes crear páginas personales y profesionales (para empresas y organizaciones). Se hace especialmente atractiva para personas que acostumbran a utilizar el resto de aplicaciones de Google porque se puede integrar cualquier gadget a golpe de un clic (Vídeos de Youtube, hojas de cálculo, diapositivas, google calendar, etc.)Además lo que más te gustará es que la herramienta es totalmente gratuita. Otra utilidad que puede ser muy interesante es que cuando se tiene un grupo de trabajo se puede centralizar todo para utilizar a site como una intranet.
